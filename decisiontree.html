2. Threshold Split: 172.5 cm#

  • Group Left (≤ 172.5)
    Data ke-6,12,3,8,14,4,10 → 7 data → semua Panthera pardus

  • Group Right (> 172.5)
    Data ke-1,9,15,5,13,2,11,7 → 8 data → semua Panthera onca

3. Entropy Awal (sebelum split)#

\[\begin{split} \begin{aligned} p_1 &= \frac{8}{15}, \quad p_2 = \frac{7}{15} \\ Entropy(D) &= -p_1 \log_2(p_1) - p_2 \log_2(p_2) \\ &= -\frac{8}{15} \log_2 \frac{8}{15} - \frac{7}{15} \log_2 \frac{7}{15} \approx \boxed{0.996} \end{aligned} \end{split}\]

4. Entropy Setelah Split#

Group Left: 7 pardus

\[ Entropy(left) = -1 \cdot \log_2 1 = \boxed{0} \]

Group Right: 8 onca

\[ Entropy(right) = -1 \cdot \log_2 1 = \boxed{0} \]

5. Information Gain#

\[\begin{split} \begin{aligned} IG &= Entropy(D) - \left( \frac{7}{15} \cdot 0 + \frac{8}{15} \cdot 0 \right) \\ &= \boxed{0.996} \end{aligned} \end{split}\]

6. Gini Index#

\[\begin{split} \begin{aligned} Gini(D) &= 1 - \left( \frac{8}{15} \right)^2 - \left( \frac{7}{15} \right)^2 \approx \boxed{0.497} \\ Gini(left) &= 0, \quad Gini(right) = 0 \\ Gini_{split} &= \frac{7}{15} \cdot 0 + \frac{8}{15} \cdot 0 = \boxed{0} \end{aligned} \end{split}\]

Visualisasi Decision Tree#

Ini adalah code program untuk mencari decision tree sekaligus rules

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np
from sklearn.tree import DecisionTreeClassifier, plot_tree, export_text
from sklearn.preprocessing import LabelEncoder

# === Data ===
data = {
    "Panjang Tubuh (cm)": [180, 200, 150, 160, 190, 140, 210, 155, 185, 165, 205, 145, 195, 158, 188],
    "Berat Tubuh (kg)": [90, 110, 60, 70, 100, 55, 120, 65, 95, 75, 115, 58, 105, 68, 92],
    "Kelas": [
        "Panthera onca", "Panthera onca", "Panthera pardus", "Panthera pardus", "Panthera onca",
        "Panthera pardus", "Panthera onca", "Panthera pardus", "Panthera onca", "Panthera pardus",
        "Panthera onca", "Panthera pardus", "Panthera onca", "Panthera pardus", "Panthera onca"
    ]
}
df = pd.DataFrame(data)

# === Label Encoding ===
le = LabelEncoder()
df['Label'] = le.fit_transform(df['Kelas'])  # tigris=0, leo=0

# === Fitur dan Target ===
X = df[["Panjang Tubuh (cm)", "Berat Tubuh (kg)"]]
y = df["Label"]  # <- gunakan label numerik hasil encoding

# === Train Decision Tree ===
clf = DecisionTreeClassifier(criterion="gini", max_depth=3, random_state=42)
clf.fit(X, y)

# === 1. Tampilkan Pohon Keputusan ===
print("\n=== Pohon Keputusan (Decision Tree) ===")
tree_rules = export_text(clf, feature_names=list(X.columns))
print(tree_rules)

=== Pohon Keputusan (Decision Tree) ===
|--- Panjang Tubuh (cm) <= 172.50
|   |--- class: 1
|--- Panjang Tubuh (cm) >  172.50
|   |--- class: 0

Visualisasi Splitting#

# === Uji 2 Data Baru ===
uji_baru = pd.DataFrame({
    'Panjang Tubuh (cm)': [200, 210],
    'Berat Tubuh (kg)': [110, 120]
})

prediksi = clf.predict(uji_baru)
hasil = le.inverse_transform(prediksi)

print("\n=== Prediksi 2 Data Baru ===")
for i, (p, b) in enumerate(zip(uji_baru['Panjang Tubuh (cm)'], uji_baru['Berat Tubuh (kg)'])):
    print(f"Data {i+1}: Panjang={p}, Berat={b} → Prediksi: {hasil[i]}")

# === Visualisasi Decision Boundary ===
print("\n=== Visualisasi Decision Boundary ===")
plot_colors = "ry"
plot_step = 1.0

x_min, x_max = X['Panjang Tubuh (cm)'].min() - 10, X['Panjang Tubuh (cm)'].max() + 10
y_min, y_max = X['Berat Tubuh (kg)'].min() - 10, X['Berat Tubuh (kg)'].max() + 10

xx, yy = np.meshgrid(np.arange(x_min, x_max, plot_step),
                     np.arange(y_min, y_max, plot_step))

grid = np.c_[xx.ravel(), yy.ravel()]
grid_df = pd.DataFrame(grid, columns=['Panjang Tubuh (cm)', 'Berat Tubuh (kg)'])
Z = clf.predict(grid_df)
Z = Z.reshape(xx.shape)

plt.figure(figsize=(8, 6))
plt.contourf(xx, yy, Z, cmap=plt.cm.RdYlBu, alpha=0.6)

# Plot titik data training
for label, color in zip([0, 1], plot_colors):
    idx = np.where(y == label)
    plt.scatter(X.iloc[idx]['Panjang Tubuh (cm)'], X.iloc[idx]['Berat Tubuh (kg)'],
                c=color, edgecolor='k', label=le.inverse_transform([label])[0])

plt.xlabel("Panjang Tubuh (cm)")
plt.ylabel("Berat Tubuh (kg)")
plt.title("Decision Boundary (Panjang vs Berat)")
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()

Data 2: Panjang=210, Berat=120 → Prediksi: Panthera onca
